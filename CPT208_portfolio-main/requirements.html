Requirements List

Three core requirements define the main interaction scope of the Maple Bridge experience.

Requirement 01

Fortune Drawing

Users should be able to draw a fortune stick and receive today’s fortune as a lightweight cultural interaction.

Requirement 02

Photo Wall

Users should be able to share their own photos and appreciate photos contributed by other visitors.

Requirement 03

Interactive Map

Users should be able to interact with the map to obtain route information, scenic spot information, and access small games.
